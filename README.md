# InetScan/ Internet Scanner
InetScann is a network utility with a web interface and API based on willdurand/Nmap wrapper, written in PHP.

### Install dependencies
`1st` get [composer](https://getcomposer.org/download/)

`2nd` Install dependencies, run a shell script called `depend.sh`
```bash
./depend.sh
```

`3rd` install [docker](https://docs.docker.com/engine/install/), if it installed you must skip this step
